## TL;DR

- **Projects** are now live in the sidebar for signed-in users.
- **Organize faster:** create projects, pick an icon and color, and group related chats.
- **Move chats freely:** send conversations into a project, switch between projects, or move them back to History.
- **Keep memory:** chats in a project can remember useful details for later chats in that same project.
- **Safe behavior:** deleting a project does not delete chats; it returns them to History.

## Introducing Projects

Projects are now live in the sidebar. Signed-in users see a dedicated **Projects** section above your prior chats.

Each project can be customized with an icon and color, then expanded to show its chats right under that project. You can:

- start new chats inside a project,
- move existing chats into a project from chat menus,
- switch chats between projects,
- remove chats back to regular chat list.

## Project Memory

Projects now include hidden memory. When a chat is inside a project, the assistant can use a short hidden summary to remember important details in future chats for that same project.

The goal is simple: keep long-running work on track without mixing details between unrelated chats.

## Safe and Transparent by Design

This release keeps things flexible and safe. Deleting a project does **not** delete chat history; it returns those chats to the regular chat list. Only you can change your projects, and project data is included when you download your account data.

This launch moves beyond one long chat list and gives teams a cleaner way to organize design, scouting, and match-day planning.
